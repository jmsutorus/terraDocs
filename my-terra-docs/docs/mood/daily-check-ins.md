# Daily Check Ins

Welcome to the **Daily Check Ins** documentation for Terra.

This section covers how to use the Mood feature to enhance your lifestyle and productivity.

## Overview

Terra's Mood module is designed to be intuitive, kinetic, and deeply integrated into your daily routine.

## Key Features

- **Seamless Integration**: Works beautifully with other Terra modules.
- **Tactile Experience**: Designed with our Earthbound Editorial philosophy.
- **Privacy First**: Your data stays yours.

## Getting Started

To get started with Mood, navigate to the Mood tab in the Terra app.
