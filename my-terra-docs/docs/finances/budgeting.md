# Budgeting

Welcome to the **Budgeting** documentation for Terra.

This section covers how to use the Finances feature to enhance your lifestyle and productivity.

## Overview

Terra's Finances module is designed to be intuitive, kinetic, and deeply integrated into your daily routine.

- **Privacy First**: Your data stays yours.

To get started with Finances, navigate to the Finances tab in the Terra app.
