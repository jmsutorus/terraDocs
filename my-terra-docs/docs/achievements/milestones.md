# Milestones

Welcome to the **Milestones** documentation for Terra.

This section covers how to use the Achievements feature to enhance your lifestyle and productivity.

## Overview

Terra's Achievements module is designed to be intuitive, kinetic, and deeply integrated into your daily routine.

## Key Features

- **Seamless Integration**: Works beautifully with other Terra modules.
- **Tactile Experience**: Designed with our Earthbound Editorial philosophy.
- **Privacy First**: Your data stays yours.

## Getting Started

To get started with Achievements, navigate to the Achievements tab in the Terra app.
