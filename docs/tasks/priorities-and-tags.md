# Priorities And Tags

Welcome to the **Priorities And Tags** documentation for Terra.

This section covers how to use the Tasks feature to enhance your lifestyle and productivity.

## Overview

Terra's Tasks module is designed to be intuitive, kinetic, and deeply integrated into your daily routine.

## Key Features

- **Seamless Integration**: Works beautifully with other Terra modules.
- **Tactile Experience**: Designed with our Earthbound Editorial philosophy.
- **Privacy First**: Your data stays yours.

## Getting Started

To get started with Tasks, navigate to the Tasks tab in the Terra app.
